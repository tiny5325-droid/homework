# 二维图形坐标变换分解工具

## 项目简介

这是一个基于PyQt/PySide6的图形化应用程序，用于计算和可视化三角形关于任意直线的反射变换。程序不仅能够计算变换结果，还能分解整个变换过程为5个基本步骤，并生成详细的矩阵计算报告。

## 核心功能

- **反射变换计算**：计算三角形关于任意直线的反射变换
- **步骤分解**：将复杂的变换分解为5个基本步骤：
  1. 平移至直线上的点
  2. 旋转至与X轴重合
  3. 关于X轴反射
  4. 旋转回原方向
  5. 平移回原位
- **可视化展示**：6个子图动态展示变换过程
- **矩阵计算报告**：详细显示每一步的变换矩阵和最终结果
- **随机数据生成**：快速生成测试数据
- **结果导出**：将计算结果保存为文本文件

## 使用说明

### 输入格式

1. **坐标输入**（A点、B点、C点）
   - 格式：`(x,y)` 或 `x,y`
   - 示例：`(1.5, -2.3)` 或 `3,4`

2. **直线表达式**
   - 格式：`ax+by+c=0`
   - 示例：`3x+5y-7=0` 或 `2x-y+1=0`
   - 支持整数和小数系数

### 操作步骤

1. **方式一**：手动输入
   - 在各输入框中填入三角形的三个顶点坐标
   - 输入反射对称轴的直线表达式
   - 点击"确定"按钮

2. **方式二**：自动生成
   - 点击"生成随机值"按钮自动填充示例数据
   - 点击"确定"按钮

3. **查看结果**
   - 左侧文本区显示详细的矩阵计算过程
   - 右侧图形区分6个子图展示变换过程
<sub>：按顺序显示初始位置和5个步骤后的三角形位置

4. **导出结果**
   - 点击"导出计算结果"将文本结果保存为.txt文件

5. **清空数据**
   - 点击"取消"按钮清空所有输入和显示内容

## 核心算法原理

### 反射变换矩阵的构成

对于关于任意直线的反射，使用以下5个基本变换的复合：

```
H = T₂ × R_inv × S × R × T₁

其中：
- T₁：平移矩阵（将直线上的点平移到原点）
- R：旋转矩阵（旋转使直线与X轴重合）
- S：关于X轴的反射矩阵
- R_inv：逆旋转矩阵
- T₂：逆平移矩阵
```

### 齐次坐标

程序使用齐次坐标进行所有变换计算，将2D点 (x, y) 扩展为3D齐次坐标 (x, y, 1)。

### calculate.py

## 常见问题

### Q: 如何输入垂直线？
A: 垂直线可表示为 `x=c` 的形式，例如 `x+0y-5=0` 表示直线 x=5

### Q: 如何输入水平线？
A: 水平线可表示为 `y=c` 的形式，例如 `0x+y-3=0` 表示直线 y=3

### Q: 坐标或表达式格式错误会怎样？
A: 程序会弹出错误提示框，说明具体的格式要求

### Q: 生成的矩阵精度如何？
A: 所有计算使用双精度浮点数，输出保留4位小数
